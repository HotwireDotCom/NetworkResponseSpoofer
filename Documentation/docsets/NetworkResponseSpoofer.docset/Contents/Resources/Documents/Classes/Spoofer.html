Spoofer

@objcMembers
public final class Spoofer : NSObject

NetworkResponseSpoofer is a network request-response recording and replaying library for iOS. It’s built on top of the Foundation URL Loading System to make recording and replaying network requests really simple.

  • normalizeQueryValues

    Enable normalizing query values, by taking only the keys and dropping the values.

    Note

    Query Value Normalization causes values (not keys) of the query parameters to be dropped while comparing URL’s. For most cases this means only one response is saved per end point if the query parameter keys are the same. Effects are
  • Reduced file size saving some storage space.
  • Consistent response for the same end point regardless of query parameter values.

    • For E.g., a url such as example.com/api?key1=value1&key2=value2 becomes example.com/api?key1&key2. This allows the Spoofer to record and replay the same reponse for all calls to the end point with similar key-value query parameters.
